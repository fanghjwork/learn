text="LOOP 2 PRINT 杨过 SPACE SPACE PRINT 小龙女 BREAK END PRINT 郭靖 SPACE SPACE PRINT 黄蓉 BREAK";
解释 text 设计

1.context  环境类
    1.定义当前字符变量 currentToken
    2.定义字符数组 StringTokenizer，（按空格拆分成多个字符）

    3.获取当前解析字符
    4.获取下一个解析字符
    5.忽略字符，获取下一个
    6.获取数字字符

2.抽象解释器
    1.解释字符
    2.执行指令

3.指令处理类，继承2
    1.定义指令集合 map
    2.解析字符，（入参1，从中获取字符）循环获取字符，判断是正常字符就创建4并调用解析字符方法，添加到map
     遇到ENd,错误指令，终止循环

    3.执行指令，循环map执行

4.指令构建类，继承2
    1.指令对象变量
    2.解析字符，判断字符是什么指令就创建什么指令对象并调用解析字符方法（创建 5/6），赋值给指令对象变量
    3.执行指令，指令对象变量调用执行方法


5.循环指令类，继承2
    1.定义循环次数
    2.定义指令对象
    3.解析字符，（入参1）忽略循环字符，获取数字数字符，赋值循环次数
    获取下一个字符，创建3并调用解析字符方法（入参1）

    3.执行指令，根据循环次数循环执行指令对象

 6.终结指令类，继承2
    1.定义指令变量 name
    2.定义输出字符变量 text
    3.解析字符，（入参1），获取当前字符保存到指令变量，忽略当前字符，获取当前字符保存到输出内容变量
    获取下一个字符
    4.执行指令，判断name是什么指令做对应的操作，（例如 name="print" 就打印输出字符变量,name="space" 就打印空格）


7.客户端
    1.定义字符变量
    2.定义环境类，（传入字符变量）
    3.创建指令处理类
    4.指令处理类调用解析字符，（传入环境类）
    4.指令处理类调用执行指令
